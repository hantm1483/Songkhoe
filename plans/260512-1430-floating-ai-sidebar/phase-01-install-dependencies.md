# Phase 01: Install Dependencies

## Context
Installing Vercel AI SDK and Anthropic SDK for the floating AI sidebar.

## Overview
- **Priority**: P1
- **Status**: pending
- **Effort**: 1h

## Requirements

Install these packages:
- `ai` - Vercel AI SDK core
- `@ai-sdk/react` - React hooks for AI
- `@anthropic-ai/sdk` - Anthropic Claude SDK

## Implementation Steps

1. Run: `npm install ai @ai-sdk/react @anthropic-ai/sdk`
2. Verify installation in package.json

## Success Criteria

- All packages installed without errors
- No version conflicts
